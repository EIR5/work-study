1. **查看git版本号** 
`git --version ` 
2. **设置用户名和邮箱 在 CMD 中运行以下命令：** 
```shell
git config --global user.name "xxx"
git config --global user.email "xxx@gmail.com"
```
3. **查看当前配置** 
`git config --list` 
4. **初始化仓库** 
`git init`
5. **克隆远程仓库**
`git clone <远程仓库地址>` 
6. **添加文件到暂存区**
`git add <文件名>`
7. **提交更改**
`git commit -m "提交备注信息"`
8. **查看状态**
`git status`
9. **推送到远程仓库**
`git push origin <分支名称>`
10. **拉取最新代码**
`git pull origin <分支名称>`
